\section{Introduction}
\par \qquad It is necessary for the WeCalEvent execution to install GlassFish server, MySQL Database to save data and JDK. 
\section{Perquisites}
\par\qquad\textbf{Install JDK 8:}\\
You can download JDK8 from this link : \\
\href{http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html}{http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html}
\par Please, choose the correct version by referring to your operation system (OS, Linux or Windows) and your system (32 bits or 64 bits).
\par\textbf{Install GlassFish 4.1:}
\par After a successful installation of JDK, download GlassFish from here: \\
\href{https://glassfish.java.net/download.html}{https://glassfish.java.net/download.html} and unzip it in a folder.
\par\textbf{Install MySQL:}\\
\qquad We need to install Database Server that will be used by our application. Please choose the right version related to your system.\\
\par\textbf{Install MySQL workbench:}\\
\qquad Finally, we need to install MySQL workbench: \href{http://www.mysql.com/products/workbench/}{http://www.mysql.com/products/workbench/} to run the data base structure needed for our application. The file will be provide in the ZIP file with our application code source.
\section{Database 	configuration}
 -	Open MySQL workbench\\
 -	File $->$ Open SQL Script $->$ Select the SQL file given with source code $->$ Execute\\
 -	Make sure Username is root and Password is empty.
